# Smart-Green-House

Sviluppo di un sistema embedded per la gestione di una serra automatizzata. Il sistema prevede la gestione automatica dei processi della serra ed inoltre la possibilità di gestione manuale del sistema, da parte dell'utente, attraverso un'applicazione Android.
Il medesimo viene realizzato a scopi didattici nell'ambito del progetto del corso di Ingegneria del Software A.A. 2015/16.
